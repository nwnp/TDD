# TDD 공부

## Installed Packages

- Jest
- node-mocks-http
- Supertest

```console
$ npm i jest --save-dev
```

```console
$ npm i node-mocks-http --save-dev
```

```console
$ npm i supertest --save-dev
```

## Running the tests

```console
$ npm test
```

## Description

- Unit test(단위 테스트), Integration test(통합 테스트)를 위한 저장소이자 공부 흔적
